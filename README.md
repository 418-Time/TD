# TD
Gra Tower defense robiona Przy pomocy Godota.
Gra Polega na bronieniu bazy w kosmosie przed nadchodzącymi przeciwnikami.

## Aktualny stan:
Wprowadzono mechaniki przeciwników bazy oraz wieżyczek narazie bez grafik.
Wprowadzono proste Ui narazie bez przeniesienia na nie zawartosci naszej figmy.
Początek tworzenia poziomów dla gry.
Obecnie głównie w fazie integracji napisanych systemów.

## Struktura plików
### Assets - Wszystkie assety naszego projektu
	Assets/Images - zdjęcia
	Assets/Texture - textury  
	Assets/Audio - audio  
		Assets/Audio/SFX
		Assets/Audio/Music

### Scenes - wszystkie sceny i skrypty
	Scenes/Entities - wszystkie jednostki w tym obiekty, postacie  
	Scenes/Maps - wszystkie mapy  
	Scenes/Menus - Wszystkie menu (gui)  
	Scenes/Global - Wszystkie globalne skrypty
